Позитивные проверки:

1. Выбрать любую страну из списка -- Отображается правильный код страны (например, +7 для России)
2. Проверить, что по умолчанию выбрана одна из стран(если предусмотрено) -- Отображается корректный код страны
3. Выбрать страну через клик мышью -- Поле кода обновляется корректно
4. Прокрутить список до конца и выбрать последнюю страну -- Страна выбирается, код отображается верно
5. Выбрать страну с несколькими вариантами кода (например, США или Россия) -- Корректное поведение (либо отображает основной код, либо предоставляет дополнительный выбор)
6. Поиск страны по названию в выпадающем списке (если есть фильтрация) -- Страна находится и выделяется при выборе
7. Изменение языка интерфейса (если поддерживается) -- Страны отображаются на нужном языке, коды остаются корректными

Негативные проверки:

Ввести несуществующее значение в поле поиска (фильтр) -- Сообщение "Ничего не найдено" или аналогичное
Попробовать выбрать пустое значение из списка -- Значение не меняется или отображается дефолтное
Открыть список и закрыть его без выбора -- Предыдущее значение сохраняется
Выполнить выбор страны в мобильном представлении -- Интерфейс адаптирован, выбор работает корректно
Попытаться выбрать страну, если список пуст (в случае ошибки загрузки) -- Отображается сообщение об ошибке или предупреждение
Перезагрузить страницу после выбора страны -- Сохранённый ранее код страны остаётся активным
Выбрать страну и проверить валидацию номера телефона ниже -- Код телефона корректно влияет на валидацию номера
